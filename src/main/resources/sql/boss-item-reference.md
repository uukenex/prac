## 강화 규칙
- 강화는 **보스처치 아이템 보상 경로**에서만 획득 가능
- 뽑기/직접구매로는 미보유 아이템만 획득 가능 (중복 불가)
- MAX_BOSS_ENHANCE = 2 (0강화, +1강화까지)
- qty 기준: qty=1 → 0강화, qty=2 → +1강화
